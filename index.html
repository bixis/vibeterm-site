// features

Real terminal

Full PTY under the hood. zsh, tmux, vim, neovim, htop — everything works. It's a real terminal, not a gimmick.

YouTube background

Paste any YouTube URL and it plays behind your terminal. Lo-fi beats, tutorials, live streams — whatever sets your mood.

Boss mode

One shortcut hides the video instantly. Your terminal looks completely normal. Your secret is safe.

Watch mode

Toggle watch mode to clear the screen and enjoy the video. Toggle back and your session is exactly where you left it.

Smart overlay

Dark highlights only on lines with text. Empty space shows the full video. Switch between row, frosted, or no overlay.

CLI controls

Control everything from the command line. Play, pause, volume, transparency — all with short commands like vibeterm p.

// commands

$ vibeterm p "https://youtube.com/..." # play video
$ vibeterm p "https://..." -a # audio only
$ vibeterm pa # pause
$ vibeterm v 40 # volume 40%
$ vibeterm w # watch mode
$ vibeterm b # boss mode
$ vibeterm t 60 # transparency 60%
$ vibeterm mode frosted # frosted glass
$ vibeterm f 18 # font size 18px

// shortcuts

Cmd+Shift+H Boss mode (hide video)
Cmd+Shift+Y URL input bar
Cmd+Shift+→ Forward 15s
Cmd+Shift+← Back 15s
Cmd+Shift+↑ Volume up
Cmd+Shift+↓ Volume down

// setup

1. Install yt-dlp (needed to fetch YouTube streams):

brew install yt-dlp

2. Download VibeTerm, open the .dmg, drag to Applications.

3. First launch: right-click > Open (macOS will warn once about unidentified developer).
